นำข้อมูลมาแสดงด้วย Template Tags

wordpress เป็นระบบจัดเก็บข้อมูลที่จะจัดเก็บข้อมูลต่างๆของเว็บไซต์ให้เรา
theme ก็จะเป็นสิ่งดึงข้อมูลพวกนั้นมาเเสดง 

ต้องการเเสดง ข้อมูลส่วนไหนของ wordpress เราก็ต้องรู้คำสั่ง

เช่นต้องการเเสดง title ของ web เราก็ต้องไปรู้คำสั่ง bloginfo('name')

ต้องเอา ตัวเเปร คำสั่ง wordpress มาเเสดง ได้เเล้ว 
<h1 class="site-title"><a href="#"><?php bloginfo( 'name' ) ?></a></h1>
//-----------

คำสั่งที่ไว้เเสดงข้อมูลคือ Template Tags หรือ คำที่ไว้เเสดงใน theme wordpress
https://developer.wordpress.org/reference/



สร้างเมนู และนำมาแสดงในธีม
ถ้าดูตรง code ก็จะตรงกับ class="site-navigation"  
เราก็ลบ div 
				<div class="menu">
					<ul>
						<li><a href="#">About</a></li>
					</ul>
				</div>
ทั้งก้อนเลย เเละใช้ คำสั่ง wordpress 

กฎ wordpress บอกว่าก่อน จะสั่ง menu ได้ ต้องประกาศ ค่ามันก่อน
สร้างไฟล์ functions.php  เเละกำหนดค่าให้มัน
